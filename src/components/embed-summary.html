Shortcode

You can generate a list containing collected entries using the following shortcode.

{% highlight html %} [contentBlock title="US Trackers" limit="2" url="http://bb.dream/betfair-predicts/us-trackers/" layout="embed_list"] {% endhighlight %}
Parameter Usage Required
title The title above the swimlane Yes
limit The number of entries you would like to display. Yes
url The url for the collection of entries (usually a blog or category) Yes
layout The layout determines how the content will be displayed. In this case 'embed_list'. Yes
