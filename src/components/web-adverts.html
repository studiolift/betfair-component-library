Container based

Adverts that respond to their container sizes. When the parent container reaches a certain size the advert will be hidden.

Supported sizes:

  • 300px
  • 468px
  • 728px

Media Query based

These adverts respond to standard media queries targeting the browser width. There are options for mobile and desktop.
