Shortcode

You can generate a grid containing collected entries using the following shortcode.

{% highlight html %} [contentBlock title="Football" limit="6" url="https://betting.betfair.com/football/" layout="grid"] {% endhighlight %}
Parameter Usage Required
title The title above the grid Yes
limit The number of entries you would like to display. Yes
url The url for the collection of entries (usually a blog or category) Yes
layout The layout determines how the content will be displayed. In this case 'grid'. Yes
date Set as 'hide' to hide the date within the entry summary. No
