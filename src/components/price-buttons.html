Shortcode

You can generate a price button using the following shortcode.

{% highlight html %} [price_button label="sbk" odds="2.5" url="https://www.betfair.com" style="lay" target="_blank"] {% endhighlight %}
Parameter Usage Required
label The button label e.g. sbk Yes
odds The button text as odds e.g. 2.5 Yes
url The url for the link Yes
style 'back' or 'lay' adjusts the background colour of the button No
target link target. Defaults to _blank No
