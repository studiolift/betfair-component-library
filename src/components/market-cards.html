Shortcode

You can generate a market card using the following shortcode.

Parameter Usage Required
text The description text for the card Yes
exc_odds Odds for the exchange button No
exc_url URL for the exchange button Only if exc_odds are added
sbk_odds Odds for the sportsbook button No
sbk_url URL for the sportsbook button Only if sbk_odds are added
